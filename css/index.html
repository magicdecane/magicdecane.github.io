overscroll-behavior

overflow-anchor
The feature is enabled by default in supporting browsers. In most cases anchored scrolling is exactly what you want — content jumping is a poor experience for anyone. Content jumping occurs for example when images above the scrolled content is loaded.

css variables

Example: 

:root {
    --eyeshare-green: #9ace37;
}

.title {
    color: var(--eyeshare-green);
}

.sub-content {
    background: linear-gradient(var(--eyeshare-green), #fff, #9cc);
}

The :root selector matches the global html document
